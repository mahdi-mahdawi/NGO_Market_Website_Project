About Breaking Bread

Breaking Bread is a online food ordering system for restaurants. Restaurant owner can add dishes via admin panel, and customers can order those dishes. It provides easy and secure checkout. Stripe payment gateway is inregrated by default.

Using Instructions

Breaking Bread is developed using CodeIgniter (v3). CodeIgniter is one of the PHP framework. 
 
#Start your localhost.
 
#Goto your root folder of the localhost.
 
#Open the application/config/config.php file with a text editor and set your base URL. If you intend to use encryption or sessions, set your encryption key.
 
#Open the application/config/database.php file with a text editor and set your database settings.
 
#Create a new database and import the database.sql found in folder DB.
 
Now you can see your application live on http://localhost:8888/DBMS/BreakingBread/

Administrator can login URL - http://localhost:8888/DBMS/BreakingBread/admin

Security

If you wish to increase security by hiding the location of your CodeIgniter files you can rename the system and application folders to something more private. If you do rename them, you must open your main index.php file and set the $system_path and $application_folder variables at the top of the file with the new name you’ve chosen.

Common security vulnerabilities: SQL injection, XSS, CSRF etc are handled by using CodeIgniter core library classes.
